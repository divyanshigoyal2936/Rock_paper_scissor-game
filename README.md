# Rock_paper_scissor-game
# Rock Paper Scissors Game 🎮

A simple **Rock-Paper-Scissors** game built using Python! 🪨📄✂️ 

## 📌 Features
- Play against the computer 🤖
- Random computer moves using `random` module 🎲
- User-friendly input validation ✅
- Win/Loss/Tie result logic 🔄
- Option to play again 🔁

## 🛠️ Requirements
- Python 3.x 🐍

## 🚀 How to Run the Game
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/rock-paper-scissors.git
   ```
2. **Navigate to the project folder**:
   ```bash
   cd rock-paper-scissors
   ```
3. **Run the Python script**:
   ```bash
   python game.py
   ```

## 🎯 How to Play
1. The game will prompt you to choose:
   - `0` for Rock 🪨
   - `1` for Scissors ✂️
   - `2` for Paper 📄
2. The computer will randomly choose one.
3. The winner is determined by standard **Rock-Paper-Scissors** rules:
   - Rock beats Scissors 🪨 > ✂️
   - Scissors beats Paper ✂️ > 📄
   - Paper beats Rock 📄 > 🪨
   - Same choices result in a tie ⚖️
4. You can choose to play again or exit.

## 📝 Example Gameplay
```
Choose one:
0 - Rock
1 - Scissors
2 - Paper
Enter your choice: 0
Computer chose: 1
You Win! 🎉
Do you want to play again? (1 for Yes / 0 for No)
```
